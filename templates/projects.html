Library Management System

This project is a comprehensive,Full stack Library Management System (LMS) designed to streamline and automate the core operations of a modern library, including cataloging, patron management, and book lending. It provides both administrative tools for library staff and a user friendly interface for patrons.

  • Allows library staff to easily add, update, and manage the entire book collection, including details like title, author, ISBN, publication year, and available copies.
  • Implemented a robust search and filtering mechanism (e.g., by author, title, or genre) for quick retrieval of any book record.
  • Functionality to register, view, and manage patron accounts,storing essential data like name, contact information. Includes logic for calculating and tracking late fees based on defined return policies.
  • Provides real time updates on the availability status of every book (e.g : Available, Checked Out, Overdue)Generates comprehensive reports on inventory levels, popular books, and overdue items to aid management decisions.
